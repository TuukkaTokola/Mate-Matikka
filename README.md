# Matematikka
# Web-projekti Ryhmä 18


## Projektin kuvaus

Projektissa toteutettiin selainpohjainen pelisivusto, jossa käyttäjä voi:
- pelata erilaisia matematiikkapelejä
- kerätä pisteitä eri peleistä
- seurata omaa kehitystään
- saada välitöntä palautetta (oikein/väärin, äänet, visuaaliset efektit)

Sivusto on suunniteltu erityisesti lapsiystävälliseksi:
- kirkkaat värit
- suuret painikkeet
- selkeä navigaatio
- välitön palaute


## Sisältö

Projektissa toteutettiin useita pelejä, joista jokaisella on oma logiikka ja vaikeustaso. 


## Pistejärjestelmä

- Jokaisesta pelistä voi saada pisteitä
- Maksimipistemäärä yhteensä: **60 pistettä**
- Pisteet tallennetaan ja näytetään erillisellä pisteet-sivulla
- Emoji-asteikko kuvaa suoritusta visuaalisesti


## Teknologiat

Projektissa käytettiin:
- HTML5
- CSS3 (Flexbox, responsive design)
- JavaScript (DOM, pelilogiikka)
- Git & GitHub (versionhallinta)
